# 行迹 - AI漫游向导 🌍

一个基于AI的沉浸式旅游探索应用，为用户提供实时生成的全景画卷和交互式探索体验。

## 产品概念

**核心理念**：将目的地"渲染"为沉浸式探索的视觉AI产品

**特色**：
- 🎨 **AI实时渲染** - 根据用户输入生成专属全景画卷
- 🔍 **沉浸式探索** - 自由缩放、平移、点击热点交互
- ✨ **个性化旅程** - 点击热点自动生成旅游攻略
- 🖼️ **原画级画质** - 多模态融合的视觉体验

## 功能特性

- ✅ 智能目的地搜索
- ✅ 快速筛选（热门/美食/城市/自然/文化）
- ✅ 全景画卷展示
- ✅ 交互式探索（缩放/平移）
- ✅ 热点信息系统
- ✅ 流畅动画效果

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

## 技术栈

- **前端**：React 18 + Vite
- **样式**：TailwindCSS + 自定义CSS动画
- **图标**：Lucide React
- **后端**：Express.js（可选）
- **AI**：Anthropic Claude API（待集成）

## 项目结构

```
src/
├── components/
│   ├── Home.jsx           # 首页（搜索+筛选）
│   └── PanoramaView.jsx   # 全景视图（交互探索）
├── styles/
│   └── index.css          # 全局样式
├── App.jsx                # 主应用
└── main.jsx               # 入口文件
```

## 用户旅程

```
搜索目的地 → AI生成全景画卷 → 用户自由探索 →
点击热点 → 查看详细信息 → 生成个性化旅程
```

## 设计规范

- **主色调**：紫色渐变 (#667eea → #764ba2)
- **辅助色**：粉色 (#EC4899)
- **字体**：系统默认字体栈
- **圆角**：20px-30px（卡片/按钮）
- **阴影**：柔和的投影效果
